    fastimage.cr

    Library that finds the dimensions and type of an image fetching as little as needed.

    Installation

    1. Add the dependency to your shard.yml:

    `yaml dependencies:

     fastimage:
   github:jetrockets/fastimage.cr

    `

    1. Run shards install

    Usage

    require "fastimage"
